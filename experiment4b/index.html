Experiment 4b - Daily Crossword

Description

This is a randomizer where the tiles display random letters to make the map look like a crossword.

Technical

I used randomness to generate the letters on top of each tile and then used modulo to get the next letter in the alphabet when you click the tile .

Reflection

This process was fun and pretty easy. Getting the tiles to be black and white was easy. The trickiest part was getting the randomized letters, but with the help of ChatGPT I was able to. I really enjoy playing crosswords so I thought that this was a fun task to try and accomplish.
